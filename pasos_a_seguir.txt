PARA USAR LA GPU
1. Debo ajustar el docker-compose.yml
services:
  yolo-detection:
    build: .
    container_name: yolo_detection_container
    volumes:
      - ./data:/app/data
      - ./output:/app/output
    environment:
      - ENV_FILE=.env
    deploy:
      resources:
        reservations:
          devices:
            - driver: nvidia
              count: 1
              capabilities: [gpu]
    runtime: nvidia
2. Instalar el runtime 
sudo apt-get install -y nvidia-container-toolkit
sudo systemctl restart docker

3. Verificar instalacion
docker run --rm --gpus all nvidia/cuda:11.8.0-base nvidia-smi

4. Al levantar el contenedor, verificar si tiene acceso
docker exec -it yolo_detection_container nvidia-smi
5. Entrenamiento  (puedo ajustar el batch hasta 32)
docker exec -it yolo_detection_container yolo train data=visdrone.yaml model=yolov10m.pt epochs=50 imgsz=640 batch=8 workers=8 device=0
